Syarat & Ketentuan

Terakhir diperbarui: Maret 2026

1. Lisensi Source Code

Setiap pembelian source code di TNCP memberikan hak kepada Anda untuk menggunakan kode tersebut untuk proyek pribadi atau komersial. Anda dilarang keras menjual kembali (resell) source code asli tanpa modifikasi signifikan atau izin tertulis.

2. Kebijakan Pengembalian Dana

Mengingat produk yang kami jual adalah produk digital yang dapat disalin, kami tidak melayani pengembalian dana (refund) setelah file dikirimkan atau akses download diberikan.

3. Dukungan Teknis

Kami memberikan dukungan instalasi dasar untuk setiap produk yang dibeli. Kami tidak bertanggung jawab atas modifikasi pihak ketiga yang menyebabkan error pada sistem.

4. Perubahan Layanan

TNCP berhak mengubah harga atau menghentikan produk sewaktu-waktu tanpa pemberitahuan sebelumnya.
